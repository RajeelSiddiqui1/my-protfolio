# **App Name**: Cyber Portfolio

## Core Features:

- Hero Section: Displays name, role, intro, CV download, and GitHub link.
- About Me: Showcase expertise, highlight achievements, and unique notes.
- Skills Grid: Interactive grid displaying various technical skills with hover animations.
- Work Experience Timeline: Vertical timeline showcasing professional history with neon accent highlights.
- Education & Projects: Minimalist cards featuring education details and personal projects with live demo buttons.
- Contact Footer: Glassmorphic footer with contact information and neon icon accents.

## Style Guidelines:

- Primary background: Dark gray (#121212) for a cyber-dark aesthetic.
- Section/Card background: Slightly lighter gray (#1E1E1E) for contrast.
- Primary text: Light gray (#E0E0E0) for readability against dark backgrounds.
- Accent highlight: Bright cyan (#4FC3F7) for interactive elements and badges.
- Body and headline font: 'Inter' sans-serif for a clean, modern look.
- Lucide Icons: Use neon-accented icons for interactive elements and contact information.
- Mobile-first flex/grid layouts for responsive design.
- Smooth scroll reveal for sections using Framer Motion.
- Card scale-up with shadow lift on hover.
- Neon glow on hover for buttons, links, and badges.
